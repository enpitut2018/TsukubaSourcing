## Elevator pitch
手伝いを気軽に行えるシステムによって人手不足を解決します。  
具体的には、仕事の登録・参加という形で、筑波大生向けの労働力の売買を実現します。

## What's
**筑波大生の筑波大生による筑波大生のためのアウトソーシングサービス**です．
